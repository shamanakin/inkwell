# Social Media Lenses

This directory contains lenses for social media content creation, specifically optimized for X (Twitter) growth.

## Lenses

**growth-lens.md** - Core growth principles and the 8 repeatable elements for non-combative X growth. Includes Reply Guy mode constraints and content-driven formatting rules.

**x-twitter-voice.md** - Platform-specific constraints and optimizations for X (Twitter). Algorithm signals, timing, formatting, and platform best practices.

## Usage

Stack these lenses with your personal voice lens:

```
Active Lenses:
1. Personal Voice Lens (theroboshaman-x-voice.md)
2. Growth Lens (growth-lens.md)
3. X Platform Voice (x-twitter-voice.md)
```

The personal voice lens defines your authentic voice. The growth lens adds optimization. The platform lens adds constraints.

---

*These lenses work together to create growth-optimized content in your authentic voice.*
