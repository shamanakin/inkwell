## How to Use

### For Batch Post Creation

1. Copy `projects/social-media-template/` to a new project
2. Use prompt from `prompts/social-media-session.md`
3. Agent loads all three lenses
4. Generate posts following content-driven formatting
5. Pattern tracking prevents formulaic repetition

### For Reply Guy Mode

**Activation:**
- Share screenshot or paste post text
- Say "reply mode" or "reply to this: [text]"
- Agent generates 3-5 reply variations
- Output is reply text only

**Quick Mode:**
- Say "quick reply" for faster workflow
- Gets 3 variations without full analysis
