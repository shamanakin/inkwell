# Social Media Content Template

This is a template project for batch creating authentic, growth-optimized X (Twitter) posts using InkWell.

## Quick Start

1. Copy this template to a new project directory
2. Update `PROJECT.md` with your specific goals
3. Load the three lenses:
   - Your personal voice lens
   - `growth-lens.md`
   - `x-twitter-voice.md`
4. Start creating posts!

## Reply Guy Mode

To activate Reply Guy mode:
- Share a screenshot or paste post text
- Say "reply mode" or "reply to this: [post text]"
- Agent will generate 3-5 reply variations
- Output is reply text only

## Structure

- `PROJECT.md` - Project manifest
- `POST-BATCH.md` - Batch planning and tracking
- `PATTERN-TRACKING.md` - Pattern tracking to prevent formulaic repetition
- `sections/` - Individual post cards
- `replies/` - Reply cards (if using Reply Guy mode)

## Key Features

- Content-driven formatting (not formulaic)
- Pattern tracking prevents AI detection
- Em dash prohibition (hard constraint)
- Natural variation through content analysis
- Growth optimization without losing authenticity
