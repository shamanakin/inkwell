# InkWell Changelog

## v0.1.0-alpha (2024-12-08)

Initial release.

### Added
- Core system architecture: lenses, projects, section cards
- `AGENT-PLAYBOOK.md`: behavioral instructions for LLM agents
- Bootstrap prompts for common workflows
- Template library for lenses, projects, and section cards
- Night's preset lenses: core, clinical, essay
- Example project: night-practice-blog
- Documentation: README, SYSTEM-DESIGN

### Philosophy
- English as code: no scripts, no CLI, just Markdown
- Agents navigate and update `.md` files as their "runtime"
- Recursive self-check pattern for long-form coherence
